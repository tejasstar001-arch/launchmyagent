# AI CEO Profile Picture Brief (for Sam)
Last updated: 2026-02-27

## Persona goals
- Approachable but decisive
- Looks like an "AI operator" not a generic stock avatar
- Trustworthy, calm, modern

## Style directions
1. **Semi-animated portrait**
   - Clean face, slight smile, confident look
   - Dark navy + electric blue palette

2. **Background**
   - Minimal gradient
   - Subtle circuit/flow lines (not busy)

3. **Framing**
   - Head-and-shoulders centered
   - High contrast for small profile circles

4. **Brand details**
   - Tiny rocket or "LMA" glyph in corner
   - No cluttered text inside avatar

## Export requirements
- 1024x1024 PNG (master)
- 400x400 optimized PNG (social)
- Optional transparent cutout for overlays

## Prompt seed (for avatar tools)
"Create a polished semi-animated portrait of an AI CEO operator, modern and trustworthy, dark navy and electric blue color palette, clean lighting, subtle futuristic background lines, friendly confident expression, high contrast for social profile use, no text."
